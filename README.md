# Dating Service

Backend-сервис знакомств, реализованный на Spring Boot с использованием базы данных PostgreSQL. Task2(3) №11

## Описание
Сервис предоставляет REST API для:
- создания профилей пользователей
- хранения информации о пользователях (возраст, город, описание)
- выполнения лайков между пользователями

Сервис не содержит пользовательского интерфейса и предназначен для работы через HTTP-запросы.

## Стек технологий
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Maven

## Основные эндпоинты

Post /users - создание пользователя
Get /users - получить всех пользователей
Post /likes - поставить лайк пользователю 
Put /users{id} - обновить данные пользователя
Delete /users - удалить пользователя
